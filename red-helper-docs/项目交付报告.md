## 📋 使用流程

### 快速开始（5分钟）

#### 1. 安装插件
```bash
1. 打开 Chrome 浏览器
2. 访问 chrome://extensions/
3. 开启「开发者模式」
4. 点击「加载已解压的扩展程序」
5. 选择 red-helper 文件夹
```

#### 2. 使用插件
```bash
1. 打开小红书网站
2. 点击浏览器右上角的红薯助手图标 🍠
3. 侧边栏自动打开
4. 选择功能（单篇笔记/博主笔记/博主信息）
5. 打开对应页面
6. 点击「开始采集」
7. 等待采集完成
8. 导出Excel或同步飞书
```

#### 3. 配置飞书（可选）
```bash
1. 点击右上角设置图标 ⚙️
2. 填写订单号、授权码、表格链接
3. 点击「保存配置」
4. 即可使用「同步飞书」功能
```
